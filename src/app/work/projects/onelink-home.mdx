## Overview

A smart home iOS application that connects users with their smoke and CO alarms equipped with embedded Amazon Alexa. As part of a 5+ developer iOS team, I contributed to the app's UI/UX improvements, Bluetooth connectivity, and device provisioning features.

## Key Features

- **Smart Alarm Management**: Real-time monitoring and control of smoke and CO detectors
- **Amazon Alexa Integration**: Voice control and smart home integration
- **CoreBluetooth Connectivity**: Reliable BLE communication with smart detectors
- **WiFi Provisioning**: Seamless device setup and network configuration
- **HomeKit Compatible**: Integration with Apple's smart home ecosystem
- **Multi-Device Support**: Manage multiple alarms across different locations

## Technologies Used

- **Swift**: Native iOS development for optimal performance
- **CoreBluetooth**: BLE communication with smart detectors
- **WiFi Provisioning**: Custom network configuration implementation
- **HomeKit**: Apple smart home framework integration
- **UIKit**: Rich, responsive user interface

My key contributions to the OneLink Home app included:

## Outcome

This smart home safety app is available on the App Store, serving thousands of users who rely on it for home safety monitoring. The app provides a seamless experience for managing smart smoke and CO alarms with voice control through Amazon Alexa integration.
