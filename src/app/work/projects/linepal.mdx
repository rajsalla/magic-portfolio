## Overview

LinePal is a consumer iOS app designed for nightlife enthusiasts, providing real-time wait times at bars, guest list management, and skip-the-line passes. Built from scratch and scaled from 0 to 50,000+ active users with a 4+ star rating on the App Store.

## Key Features

- **Real-time Wait Times**: Live updates on bar wait times to help users make informed decisions
- **Guest Lists & Skip-the-Line**: Integrated guest list system and premium skip-the-line passes
- **Liquid Glass UI**: iOS 26 ready with modern Liquid Glass design language
- **High Performance**: Sub-200ms API response times with 99.6% crash-free sessions
- **Seamless UX**: Built with SwiftUI using MVVM architecture for maintainable, scalable code

## Technologies Used

- **Swift & SwiftUI**: Modern iOS development with declarative UI
- **MVVM Architecture**: Clean separation of concerns for maintainability
- **Django Backend**: RESTful API backend for data management
- **Firebase**: Analytics, crash reporting, and push notifications
- **TestFlight**: Beta testing and continuous deployment

## Challenges and Learnings

Building a consumer app from zero to 50,000+ users presented unique challenges in scaling, performance optimization, and user experience. Key learnings included:

- Optimizing API calls and implementing efficient caching strategies to achieve sub-200ms response times
- Building robust error handling to maintain 99.6% crash-free sessions
- Designing intuitive UI/UX that resonates with the nightlife audience
- Managing app store presence and user feedback to maintain 4+ star rating

## Outcome

LinePal successfully launched on the App Store and grew to over 50,000 active users. The app maintains a 4+ star rating with 99.6% crash-free sessions, demonstrating both technical excellence and product-market fit. The app is iOS 26 ready with modern design patterns.
